Flor Amarilla — Ábrela con un link

Guarda este archivo (index.html). Si lo abres con ?open=1 en la URL la flor aparecerá abierta automáticamente — útil para compartir en un link.

Consejo: sube el archivo a tu hosting (Netlify, GitHub Pages, Vercel) y comparte ?open=1 para que el receptor vea la flor abierta automáticamente.
